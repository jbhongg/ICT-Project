# ICT-Project
설비의 데이터를 수집하여 실시간 데이터 시각화 및 시간에 따라 수집된 데이터 확인, 날짜별 데이터 검색, 데이터 예측, 데이터별 상관관계,  안정도 측정 기능을 제공해주는 플랫폼입니다.
